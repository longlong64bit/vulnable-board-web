게시판 스트리밍

게시판 콘텐츠를 React Server Components(RSC) 스트림으로 제공합니다. 모바일 앱·외부 연동 시 사용합니다.

로그인한 상태에서 게시판 목록·글을 조회할 때, 서버가 RSC 스트림으로 응답합니다. 아래 API로 직접 스트림을 요청할 수 있습니다.

연동 API

스트림 렌더

POST /api/rsc/render — 게시판에서 사용하는 RSC 스트림을 역직렬화합니다. Body: Content-Type: text/x-component

스트림 재생

POST /api/rsc/replay — 수신한 스트림을 서버에서 재생(검증용). Body 형식은 위와 동일합니다.

외부 시스템 연동·테스트 시 위 엔드포인트를 POST로 호출하세요. GET으로 접속하면 이 안내 페이지로 이동합니다.
